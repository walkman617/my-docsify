初始化应用

创建一个简单的、基于模板的 Flutter 工程，将项目命名为 form ，接下来你将会修改这个工程来完成最终的 App。

下面这个 GIF 可以引导你预览本 codelab 做完之后的应用效果图：

在这个步骤中，你将主要编辑 Dart 代码所在的 lib/main.dart 文件：

删除 lib/main.dart 中的所有代码，然后替换为下面的代码，运行代码将显示一个标题为 Form 的空白页面。 


import 'package:flutter/material.dart';
void main() => runApp(MyApp());
class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: FormPage(),
    );
  }
}

class FormPage extends StatefulWidget {
  @override
  State createState() => FormState();
}

class FormState extends State {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text("Form"),
      ),
    );
  }
}

添加两个标签页

在这一步中，将在页面中添加一个包含两个标签的标签栏，以便于在接下来的步骤中，在第一个标签页中布置用户信息表单，在第二个标签页中布置系统设置控件。

将页面修改为 DefaultTabController 组件，修改 FormState 类中的 build 函数： 


class FormState extends State {
  @override
  Widget build(BuildContext context) {
    return DefaultTabController(
      initialIndex: 0,
      length: 2,
      child: Scaffold(
        appBar: AppBar(
          title: const Text('TabBar Widget'),
          bottom: const TabBar(
            tabs: [
              Tab(
                child: Icon(
                  Icons.face,
                  color: Colors.white,
                ),
              ),
              Tab(
                child: Icon(
                  Icons.settings,
                  color: Colors.white,
                ),
              ),
            ],
          ),
        ),
        body: TabBarView(
          children: [
            SingleChildScrollView(
              padding: EdgeInsets.all(10),
              child: Text("Information"),
            ),
            SingleChildScrollView(
              padding: EdgeInsets.all(10),
              child: Text("Setting"),
            ),
          ],
        ),
      ),
    );
  }
}

添加音量调节

设置音量变量，在 FormState 类中设置变量 _volume ： 


double _volume = 50;

添加音量控制组件 Slider ，修改 FormState 类中的 build 函数，在通知开关组件 ListTile 之后添加组件： 


ListTile(
  title: Text("Volume"),
  subtitle: Slider(
    value: _volume,
    min: 0,
    max: 100,
    divisions: 20,
    label: _volume.round().toString(),
    onChanged: (double value) {
      setState(() {
        _volume = value;
      });
    },
  ),

添加权限开关

分别设置存储、位置权限的变量，在 FormState 类中设置变量 _storage 和变量 _location ： 


bool _storage = true, _location = false;

添加权限复选组件 Checkbox ，修改 FormState 类中的 build 函数，在音量组件 ListTile 之后添加组件： 


ListTile(
  title: Text("Permission"),
  subtitle: Column(
    children: [
      ListTile(
        title: Text("Storage"),
        leading: Checkbox(
          value: _storage,
          onChanged: (bool? value) {
            setState(() {
              _storage = value as bool;
            });
          }
        )
      ),
      ListTile(
        title: Text("Location"),
        leading: Checkbox(
          value: _location,
          onChanged: (bool? value) {
            setState(() {
              _location = value as bool;
            });
          }
        )
      )
    ],
  ),
),

添加模式选择

设置模式枚举类型，在全局添加枚举类型变量 Pattern ： 


enum Pattern { easy, hard }

设置模式控制变量，在 FormState 类中设置变量 _pattern ： 


Pattern _pattern = Pattern.easy;

添加模式单选组件 Radio ，修改 FormState 类中的 build 函数，在权限组件 ListTile 之后添加组件： 


ListTile(
  title: Text("Pattern"),
  subtitle: Column(
    children: [
      ListTile(
        title: Text("Easy"),
        leading: Radio(
          value: Pattern.easy,
          groupValue: _pattern,
          onChanged: (Pattern? value) {
            setState(() {
              _pattern = value as Pattern;
            });
          },
        ),
      ),
      ListTile(
        title: Text("Hard"),
        leading: Radio(
          value: Pattern.hard,
          groupValue: _pattern,
          onChanged: (Pattern? value) {
            setState(() {
              _pattern = value as Pattern;
            });
          },
        ),
      )
    ],
  ),
),
