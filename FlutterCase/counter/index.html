初始化应用

创建一个简单的、基于模板的 Flutter 工程，将项目命名为 counter，接下来你将会修改这个工程来完成最终的 App。

下面这个 GIF 可以引导你预览本 codelab 做完之后的应用效果图：

在这个步骤中，你将主要编辑 Dart 代码所在的 lib/main.dart 文件：

删除 lib/main.dart 中的所有代码，然后替换为下面的代码，运行代码将显示一个标题为 Counter 的空白页面。 


import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: CounterPage(),
    );
  }
}

class CounterPage extends StatefulWidget {
  @override
  State createState() => CounterState();
}

class CounterState extends State {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text("Counter"),
      ),
    );
  }
}
